## Como usar

Faça o clone deste repositório em sua máquina.

```bash
    git clone https://github.com/Ramiro-Rfn/Desafio01-ignite.git

````

Instale as dependências rodando:
```bash
    yarn 

    //ou

    npm install
````

Rode o progecto:
```bash
    yarn dev
